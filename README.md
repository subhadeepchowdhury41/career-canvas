How to run:

1. Clone the repository
   -- git clone https://github.com/subhadeepchowdhury41/career-canvas.git

2. Install dependencies
   -- cd career-canvas
   -- npm install

3. Set up environment variables
   -- cp .env.example .env
   -- Edit .env file with your environment variables. (Add Neon database connection URL)

4. Run the server
   -- npm run dev

5. Open the client in your browser
   -- Open http://localhost:5000 in your browser
